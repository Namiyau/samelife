# SameLife V0.63 样本详情弹窗

V0.63 增加食品样本详情弹窗。

## 功能

点击结果页中食品小项的单价，可以看到：

- 当前采用单价
- 数据源口径
- summary 样本数
- summary 计算方法
- 样本统计：中位数、平均数、最低价、最高价、样本数
- 原始样本表
  - 样本名称
  - 价格
  - 来源名称
  - 来源链接
  - 日期
  - 地区
  - 是否含配送费
  - 是否含服务费
  - 是否含优惠
  - 备注
  - dataQuality

## 限制

目前只有食品类接入了原始样本 JSON。
交通、住房、娱乐、日用品仍然是占位数据，所以点击这些小项时只显示基础来源，不显示样本表。
