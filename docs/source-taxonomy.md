# SameLife 数据源定义 V0.53

## SL官方

SL官方不是“只复制统计局平均值”，而是指 SameLife 项目维护的公开可核查数据集。

可包含：

- 官方统计
- 官方票价
- 企业公开菜单价
- 企业公开套餐价
- 平台公开标价

原则：

- 不考虑临时优惠券
- 不考虑需要特殊会员资格的折扣
- 不考虑很短期的活动价
- 如需要二次处理，必须写清计算方法，例如平均数、中位数、样本数、采样区域

## SL综合

SL综合在 SL官方基础上加入真实生活体验修正。

可包含：

- 常驻优惠券
- 常见会员价
- 外卖配送费
- 平台服务费
- 长期存在的套餐组合
- 实际用户更可能购买的规格

原则：

- 不纳入极端薅羊毛价格
- 不纳入一次性新人券
- 不纳入难以复现的个体优惠
- 必须写清“优惠是否常驻”和“是否含配送/服务费”

## NUMBEO

NUMBEO 作为外部众包参考源，适合对照，不建议作为 SameLife 的唯一结论来源。

## 问卷小项原则

三种数据源应尽量保持同一套类别和 item id。切换数据源时，最好只改变价格和口径，不改变用户要填写的问题。
