# SameLife 数据源口径 V0.51

V0.51 在每月频率页新增数据源选择：

- SL官方：本项目寻找并制作的尽量官方/准官方数据。
- SL综合：综合官方价格、实际体验、常驻优惠等因素的 SameLife 综合数据。
- NUMBEO：完全使用 Numbeo 口径的数据版本，适合作为众包参考。

## 类别是否应随数据源变化？

建议尽量不要变化。

原因：SameLife 的核心是“同一套生活方式”在不同城市、不同数据口径下的成本比较。如果切换数据源后类别和小项大幅变化，结果就不再可比。

推荐规则：

1. 所有数据源尽量使用同一套类别和商品 ID。
2. 某个来源缺少某个商品时，显示“暂无数据”或使用备用来源，不直接删除问卷项。
3. 结果页标记该小项使用的具体数据源。
4. 未来可以提供“严格模式”：只计算所有数据源都覆盖的小项。
