# SameLifeV0.64g

## 更新内容

- 食品数据接入文件改为 `src/data/2026_06_food.json`
- 当前版本只接入 SL官方；SL综合和 Numbeo 显示为待接入，不再使用旧占位数据替代
- 结果总额显示对方货币括号换算值，并移除“xx原币”行
- 商品篮子详情顶部“来源名称”改为纯文本，不再作为超链接
- 原始样本明细新增“可信度”列
- 分类结果的“来源可信度”改为 5 星平均分

## 可信度规则

- 5星：可确认事实信息，例如官网、官方菜单、官方商品页
- 4星：手动计入，且经公开来源范围核查认为基本合理
- 3星：手动计入，但仍需 App/门店/截图确认
- 2星：新闻、自媒体、第二手转述
- 1星：推测信息
