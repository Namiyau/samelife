# SameLife V0.6 商品篮子计算方案

V0.6 的用户输入采用“生活场景频率”，后台仍然需要维护具体商品价格。

## 食品

### 自己做饭一餐
建议后台篮子：
- 主食：米饭/面/面包
- 蛋白质：鸡蛋、鸡肉/猪肉/鱼/豆腐
- 蔬菜：绿叶菜、番茄/土豆/洋葱等
- 调味与油：按每餐分摊
- 能源：燃气/电按每餐分摊，可先忽略或低权重

计算建议：按 7 天普通菜单估算，再除以餐数，得到单餐成本。

### 便利店/超市即食一餐
样本：
- 便当
- 饭团/三明治
- 沙拉/热食
- 饮料可单独或不含

计算建议：取 5-10 个样本中位数。

### 普通外食一餐
样本：
- 快餐
- 定食/盖饭/面
- 普通工作日午餐

计算建议：排除高档餐厅，取中位数。

### 外卖一餐
样本：
- 商品价
- 配送费
- 平台服务费
- 常驻优惠

SL官方：不含临时优惠。
SL综合：可含常驻优惠、会员常见优惠和配送费。

## 交通

- 地铁/电车单程：官方票价或典型通勤距离票价
- 公交单程：官方票价
- 出租车/网约车短途：典型 3-5 公里样本
- 通勤月票/定期券：典型通勤区间
- 共享单车/自行车一次：平台常规价

## 住房

- 单人租房月租：非豪宅、普通通勤区间、固定面积段
- 合租/房间月租：单间或 share house 样本
- 水电燃气：单人月度样本
- 家庭网络：普通宽带套餐
- 手机套餐：普通流量套餐
- 物业/管理费：有则填，无则为 0 或独立说明

## 娱乐

- 电影：普通成人票
- 健身房：普通月卡
- 流媒体：常用会员月费
- 展览/博物馆：普通展览/门票
- KTV/Live/活动：普通一次消费
- 游戏/数字内容：月度订阅或常见小额消费

## 日用品

- 洗发水/沐浴露：常规容量，按购买一次计
- 纸巾/卫生纸：常规家庭装/组合装
- 洗衣液/清洁用品：洗衣液、洗洁精、清洁剂可合并
- 护肤/化妆品：基础护肤品，不含高端奢侈品
- 理发：普通理发一次
- 常备药/药妆：感冒药、止痛药、创可贴、维生素等普通样本
